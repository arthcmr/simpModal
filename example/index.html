simpModal

simpModal is a simple modal jquery plugin based on leanModal by FinelySliced (See their website). I built simpModal because I wanted flexibility. Many of the modal plugins I've seen out there rely on lots of HTML markup, which makes them too complex for simple things. Plus, I wanted to be able to code my CSS separately from the plugin, which certainly makes it a lot more designer-friendly. Finally, I wanted the plugin to add/destroy any necessary HTML tags it may need, so that I wouldn't have unnecessary tags when a modal isn't visible.

Best things about this plugin so far:

  • Simple
  • Flexible width and height
  • Independent CSS
  • Ajax Support
  • Image Free
  • Very light (less than 4Kb)
  • Multiple instances on the same page
  • Many options
  • Very easy to use

Yet to be improved:

  • No support for old IE browsers
  • No iframe support
  • No gallery support

Usage

Using simpModal is quite simple and you can start with something as simple as this:

$.simpModal({
        content: "Simple Message"
        });

However, there are plenty of options you may find useful:

$.simpModal(options);

Default options:

$.simpModal({
	top: 100,
	overlay: 0.4,
	extraClass: null,
	element: null,
	link: null,
	file: null,
	params: null,
	method: 'post',
	content: null,
	width: null,
	closeButton: false,
	loadingMessage: "Loading...",
	speed: 200,
	transition: 'fade',
	clickEnter: true,
	closeEsc: true,
	close: false
	});

Examples

Close Button

$.simpModal({
        content: "Duis autem vel eum iriure dolor in hendrerit in vulputate velit esse molestie consequat, vel illum dolore eu feugiat nulla facilisis.",
        closeButton: true
        });

Extra Class

You may apply any class to your modal so that you can have multiple modal styles

$.simpModal({
        ...
        extraClass: 'narrow'
        });

Overlay transparency

$.simpModal({
        ...
        overlay: 1
        });

Load contents from another element (hidden or not)

This div's id is "element". Type anything you want here and click the button below. The modal will contain these words =)
$.simpModal({
        ...
        element: $("#element")
        });

Load file (ajax)

It supports AJAX!

$.simpModal({
        ...
        file: "file.html",
        loadingMessage: "Loading a file: file.html"
        });

Custom size

$.simpModal({
        ...
        width: "487px"
        });

Control animation speed

$.simpModal({
        ...
        speed: 2000
        });

Choose your transition

'fade' is the default transition but you may choose another one. Currently supported transitions: fade, slide, slideUp, slideRight. More to come.

$.simpModal({
        ...
        speed: 400,
        transition: 'slide'
        });

Keypress events: Enter and Esc keys

What happens when you the Enter or Escape keys are pressed? The modal may close, form may be submitted or something else may happen: it's up to you. By default, Esc closes the modal and Enter "clicks" the submit button (when there is one)

$.simpModal({
        ...
        enterButton: "first" //"click" the first button when Enter key is pressed
        });

Close modal buttons

Adding the attribute data-close="true" to any tag creates a link that closes the modal when clicked

$.simpModal({
        ...
        enterButton: "first"
        });

How do you want to close it? Click outside, click on the "Cancel" button, hit the Escape Key, hit ENTER, or click here

Close modal programmatically

Sometimes we just want to close a modal automatically or in some other circumnstances. In that case, just do this:

$.simpModal({ close: true });
