simpModal

simpModal is a simple modal jquery plugin based on leanModal by FinelySliced (Visit their page). I built simpModal because I wanted flexibility. Many of the modal plugins I've seen out there rely on lots of HTML markup, which makes them too complex for simple things. Plus, I wanted to be able to code my CSS separately from the plugin, which certainly makes it a lot more designer-friendly. Finally, I wanted the plugin to add/destroy any necessary HTML tags it uses, so that I wouldn't have unnecessary tags flying around my code when a modal isn't visible.

Using simpModal is quite simple and you can start with something as simple as this:

$.simpModal({ content: "Simple Message" });

Features

  • Simple
  • Flexible width and height
  • Independent CSS
  • Ajax Support
  • Image Free
  • Very light (less than 4Kb)
  • Multiple instances on the same page
  • Many options
  • Very easy to use

Options

There are many options, but not enough to make it confusing. It's actually quite simple to use them. Take a look at the default options:

$.simpModal({
    top: 100,
    overlay: 0.4,
    extraClass: null,
    element: null,
    link: null,
    file: null,
    params: null,
    method: 'post',
    content: null,
    width: null,
    closeButton: false,
    loadingMessage: "Loading...",
    speed: 200,
    transition: 'fade',
    clickEnter: true,
    closeEsc: true,
    close: false
    });

Examples

Show Close Button

You may want to display a nice looking button:

$.simpModal({
        content: "Duis autem vel eum iriure dolor in hendrerit in vulputate velit esse molestie consequat, vel illum dolore eu feugiat nulla facilisis.",
        closeButton: true
        });

Extra Class

You may apply any class to your modal so that you can have multiple modal styles

$.simpModal({
        ...
        extraClass: 'narrow'
        });
